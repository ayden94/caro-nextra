# Caro-kann을 소개합니다

caro-kann은 내부적으로 useSyncExternalStore 훅을 사용해 작성된 전역 상태 관리 도구입니다. 타입스크립트를 지원하며 Next.js와 React.js에서 사용할 수 있습니다. useState와 유사한 문법을 채택하고 있기에 React.js에 익숙한 개발자라면 누구나 쉽게 사용할 수 있습니다.

## caro-kann@3.1.0에서 새로 추가된 것

* 새로운 버전의 caro-kann에는 다양한 미들웨어가 추가되었습니다. logger, validate, debounce 등의 미들웨어를 통해 더욱 유연한 상태 관리를 지원합니다.
* 이제 zustand 미들웨어는 deprecated 되었습니다. 미들웨어 합성에 사용할 수 없으며 유지 보수 되지 않습니다. 하지만 하위 호환성은 여전히 유지됩니다.
* persist 미들웨어의 migrate 로직이 변경되었습니다. 이제 migrate pipe를 통해 안정적이고 효과적으로 persist versioning을 처리할 수 있습니다.

## 설치하는 법

Caro-Kann은 아래의 다양한 방법으로 설치할 수 있습니다.

```bash
npm i caro-kann@latest
pnpm add caro-kann@latest
yarn add caro-kann@latest
bun add caro-kann@latest
```
